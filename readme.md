# Client DLL

## Dependencies
- Microsoft Detours 4.x  
  - The project is configured to look for Detours under `external/detours`.
  - Directory tree:
    - `external/detours/include`
    - `external/detours/lib.X86` (for Win32)
    - `external/detours/lib.X64` (for x64)

## Build
1. Open this folder in Visual Studio (`Client.sln`).
2. Select **Release** and **Win32/x86** configurations.
